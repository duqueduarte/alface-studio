# Repol Record 🎥

Um gravador de tela simples feito em **Python**, com suporte a áudio e vídeo.  
Este projeto foi criado para facilitar gravações rápidas e pode ser expandido para edição básica.

## 📦 Download do aplicativo
Se você não quiser compilar o código, pode baixar o executável pronto aqui:  
[⬇️ Baixar Alface Studio](https://drive.google.com/file/d/1VsL-iCI5q2tPzkROpfbJLt325aixwJNz/view?usp=drive_link)

## 🚀 Como rodar pelo código

### Pré-requisitos
- Python 3.8 ou superior
- pip (gerenciador de pacotes Python)

### Instalação
Clone o repositório:
```bash
git clone https://github.com/duqueduarte/alface-studio.git
cd alface-studio
```

Instale as dependências:
```bash
pip install -r requirements.txt
```

### Execução
Execute o script principal:
```bash
python Software_gravacao.py
```

## 📋 Funcionalidades
- ✅ Gravação de tela (screencast)
- 🔊 Captura de áudio
- 📹 Suporte a vídeo
- ⚡ Interface simples e intuitiva
